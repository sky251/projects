Add Products in order line in selected SO and PO from Wizard.

This module will help Partner to add multiple products in order line in selected SO and PO from wizard.

Prerequisites before Installation

No Prerequisites

Select SO and in action button click on "Select Product"

Select Products and click on Update.

Selected Products is added in order line in Selected SO.

Select PO and in action button click on "Select Product"

Select Products and click on Update.

Selected Products is added in order line in Selected PO.
