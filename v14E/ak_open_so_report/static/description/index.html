Open Sale Order Report /Not delivered Sale orders

This module allow customer to get report of Sale order which are not delivered yet.

Prerequisites before Installation

Install requirement.txt in your system.

Go to Reporting Menu for Open Sale Order Report

Select the duration and partners and there you go.

Open Sale Order PDF report filtered by Partner.

Open Sale Order XLSX report filtered by Partner.

Select the duration and Categories and print.

Open Sale Order PDF report filtered by Product Category.

Open Sale Order XLSX report filtered by Product Category.
